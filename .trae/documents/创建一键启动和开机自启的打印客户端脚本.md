# 创建一键启动和开机自启的打印客户端脚本

## 方案目标
- ✅ 双击一键启动打印客户端
- ✅ 后台无窗口运行
- ✅ 开机自动启动
- ✅ 详细的日志记录
- ✅ 自动重连机制

## 实现步骤

### 1. 创建启动脚本
- **`start-print-client.bat`**：批处理文件，用户双击启动
- **`run-print-client.vbs`**：VBScript文件，实现无窗口后台运行
- **`add-startup.reg`**：注册表脚本，添加开机自启

### 2. 增强打印客户端
- 修改`print-client.js`，添加日志文件输出功能
- 优化错误处理和重连机制
- 添加进程管理和状态监控

### 3. 配置文件优化
- 更新`print-client-config.json`，添加日志路径配置
- 支持更多自定义选项

### 4. 部署文档
- 创建`README.md`部署指南
- 提供详细的使用说明

## 文件结构
```
scripts/
├── print-client.js          # 主客户端脚本
├── print-client-config.json # 配置文件
├── start-print-client.bat   # 启动批处理文件
├── run-print-client.vbs     # 后台运行脚本
├── add-startup.reg          # 开机自启注册表脚本
└── README.md                # 部署指南
```

## 技术特点
- **零依赖**：不需要安装额外软件
- **简单易用**：用户只需双击操作
- **稳定可靠**：后台运行，不受用户操作影响
- **可维护性**：详细的日志记录便于排查问题
- **跨版本兼容**：支持不同版本的Windows系统

## 部署流程
1. 将所有文件复制到目标电脑
2. 双击`add-startup.reg`添加开机自启
3. 双击`start-print-client.bat`启动服务
4. 检查日志文件确认服务运行正常

这样，用户就可以通过简单的点击操作启动打印客户端，并且系统会在开机时自动运行，无需手动干预。
