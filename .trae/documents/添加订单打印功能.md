## 实现订单打印功能

### 目标
在管理后台的订单管理页面添加打印按钮，实现与用户提供的txt文档完全一致的打印格式。

### 实现步骤

1. **修改 OrderManage.vue 组件**
   - 在订单状态列中添加打印按钮
   - 为每个订单行添加打印功能

2. **实现打印函数**
   - 创建 `printOrder` 函数，接收订单对象作为参数
   - 生成与txt文档格式完全一致的打印内容
   - 包含：标题、订单号、商品列表、备注、时间等信息
   - 使用固定宽度字体和精确的空格对齐

3. **打印样式设计**
   - 设置打印页面样式，确保边距和字体大小正确
   - 使用等宽字体（如 Courier New）确保对齐一致
   - 调整打印预览设置以匹配用户的期望

4. **测试验证**
   - 测试打印功能，确保输出与txt文档格式完全一致
   - 验证不同订单数据的打印效果

### 技术实现要点

- 使用 `window.print()` API 触发打印
- 创建临时打印区域，包含格式化的订单内容
- 应用专用的打印CSS样式
- 确保中文显示正常，无乱码问题
- 保持与用户提供的txt文档完全一致的布局和格式
