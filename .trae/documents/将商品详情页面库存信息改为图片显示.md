## 任务目标
将商品详情页面中的库存信息从文本显示改为图片显示，与参考图片中的样式保持一致。

## 实现步骤

### 1. 修改 ProductDetail.vue 文件
- **目标**：将库存信息的文本显示改为图片显示
- **位置**：`ProductDetail.vue` 文件中的 `stock-info` 部分（第 44-48 行）
- **修改内容**：
  - 将库存数量文本替换为图片显示
  - 保持已售数量的显示
  - 确保样式与参考图片一致

### 2. 实现图片显示逻辑
- **方案**：使用动态生成的图片 URL 来显示库存信息
- **实现方式**：
  - 使用 `text_to_image` API 生成包含库存数量的图片
  - 构建合适的 prompt 来生成符合设计风格的图片
  - 设置适当的图片尺寸和格式

### 3. 优化样式和响应式设计
- **目标**：确保图片显示与页面其他元素协调一致
- **实现内容**：
  - 调整图片大小和位置
  - 确保在不同设备上的显示效果
  - 保持与页面整体设计风格的统一性

### 4. 测试和验证
- **目标**：确保修改后的功能正常工作
- **测试内容**：
  - 验证库存信息正确显示为图片
  - 测试不同库存数量的显示效果
  - 确保页面加载性能不受影响

## 技术要点
- 使用 Vue 的动态绑定来生成图片 URL
- 利用 `text_to_image` API 生成自定义图片
- 保持代码的可维护性和可读性
- 确保修改后的页面美观且响应式
